README - Instrukcja uruchomienia projektu

1. Rozpakuj folder z archiwum ZIP na swoim komputerze.

2. Otwórz Visual Studio Code.

3. W lewym górnym rogu kliknij "File" (Plik) -> "Open Folder" (Otwórz folder).

4. Wybierz rozpakowany folder z projektem i potwierdź.

5. Po załadowaniu projektu w Visual Studio Code, naciśnij klawisz F5, aby uruchomić debugowanie i start aplikacji.

6. Po uruchomieniu aplikacji, otwórz przeglądarkę internetową i wpisz adres:
   http://localhost:5200/

7. Zaloguj się używając danych:
   - Login: admin
   - Hasło: admin

8. Po poprawnym zalogowaniu zobaczysz komunikat o pomyślnym logowaniu.

9. Co możesz teraz zrobić w aplikacji:
   - Obserwować aktualne kursy walut na stronie.
   - Wygenerować losowy obrazek pieska, klikając odpowiedni przycisk.
   - Dodawać nowe posty, edytować istniejące lub usuwać je za pomocą integracji z serwisem JSONPlaceholder.

Dziękujemy za skorzystanie z aplikacji!
W razie pytań lub problemów zapraszamy do kontaktu.
